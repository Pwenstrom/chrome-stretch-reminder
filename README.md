# Resningspåminnelse – Chrome Extension

Ett enkelt Chrome-tillägg som påminner dig att resa dig och röra på dig under arbetsdagen.

![Tillägg-ikon](icons/icon128.png)

## Funktioner

- Påminner dig med en systemnotis med jämna mellanrum
- Valbart intervall: 15, 30, 45 eller 60 minuter — eller eget värde
- Snooza påminnelsen 5 minuter om du inte kan resa dig just nu
- Pausa påminnelserna när du inte behöver dem
- Räknar hur många gånger du rest dig idag
- Tre karaktärer att välja mellan (klicka på figuren för att byta)
- Hoppar automatiskt över påminnelse om datorn varit inaktiv

## Installation

Tillägget finns ännu inte i Chrome Web Store och installeras manuellt:

1. Ladda ner eller klona det här repot
2. Öppna Chrome och gå till `chrome://extensions`
3. Aktivera **Utvecklarläge** (toggle uppe till höger)
4. Klicka **Läs in okomprimerat tillägg**
5. Välj mappen där du lade filerna
6. Tillägget visas nu i verktygsfältet

## Användning

Klicka på tilläggsikonen i verktygsfältet för att öppna popup-fönstret där du kan:

- Se hur lång tid det är till nästa påminnelse
- Ändra intervall
- Pausa och starta om påminnelserna
- Byta karaktär

## Krav

- Google Chrome (eller Chromium-baserad webbläsare)
- Windows: se till att Chrome har tillstånd att visa aviseringar under **Inställningar → System → Aviseringar**
